Android Studioでデフォルト作成プロジェクトでビルドエラー (解決)

Posted in troubleshoot with tags Android Studio troubleshoot - Tuesday, January 28, 2014

デフォルト作成プロジェクトでビルドするとエラーが出た。

Failed to refresh Gradle project ‘My Application’ Could not find any version that matches com.android.support:appcompat-v7:+. Required by: MyApplication:app:unspecified Please install the Android Support Repository from the Android SDK Manager.

SDK Managerから Extra -> Android Support Repository をインストールして， Rebuildして解決。

IDE: Android Studio 0.5.2
