npmをインストールしようとすると301でリダイレクトされる (解決)

Posted in troubleshoot with tags troubleshoot nodejs - Tuesday, July 8, 2014

curl http://npmjs.org/install.sh > install.sh だと，301のhtmlが保存されてしまうので，下記コマンドで対応した。

curl -O -L -k <https://npmjs.org/install.sh> \> install.sh
