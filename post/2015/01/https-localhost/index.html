https://localhost/ にアクセス出来ない (解決)

http://localhost/ には接続できるが，httpsには接続できない問題が発生。 調査方法からメモしておく。

  • 443ポートに接続できるかどうか

sudo lsof -i tcp:80 -i tcp:443 | grep LISTEN

  • ファイアウォールが動いていないか システム環境設定 > セキュリティとプライバシー > ファイアウォールを確認

  • httpd.conf の設定が正しいか

  • httpd-ssl.conf の設定が正しいか

$ sudo /usr/sbin/httpd -k start -e Debug -E /dev/stdout

Yosemiteでapache2のバージョンが上がったことで，今までの設定ファイルでは動かなかった模様。 モジュールの読込が足りなかったことと，mutexの設定をコメントアウトして解決。

変更したファイル * /private/etc/apache2/httpd.conf * /private/etc/apache2/extra/httpd-ssl.conf
