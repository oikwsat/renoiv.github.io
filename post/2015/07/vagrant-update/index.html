Vagrantを1.7.2から1.7.4にupdate

Posted in develop with tags vagrant memo - Friday, July 31, 2015

使用環境

  • Mac OS X Yosemite 10.10.4
  • Vagrant 1.7.2

アップデート方法

  1. https://www.vagrantup.com/downloads.html にアクセスしダウンロード
  2. 解凍してVagrant.pkgからインストール
$ vagrant version
Installed Version: 1.7.4
Latest Version: 1.7.4

You're running an up-to-date version of Vagrant!
