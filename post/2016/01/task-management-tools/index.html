2015年に使ったタスク管理ツールまとめ

Posted in tool with tags tool lifehack - Tuesday, January 5, 2016

2016年あけましておめでとうございます。
新年ということで今まで使ってきたタスク管理ツールを比較・ふりかえり， タイムマネジメントを見直していきたいと思います。

1. Any.do

プライペート用に1年近く，iPhone/Androidアプリで使ってきました。
UIがかっこいいという理由から使い始めたのですが，プライベート用ということもあり惰性で使っていただけで，使いこなすに至りませんでした。

2. Doit.im

仕事用に，Mac版アプリで1年ほど使いました。
マネジメント業務が多くなり，差し込みも多くなってきてタスクを落としがちだったということで導入。
無料版だと同期が1回/日しかできない＆Mac版アプリが提供されないので有料版で使ってましたが，同僚とのタスク共有をやりたかった関係でTodoistへ移行。

UIが若干ビジネス寄りではありますが，繰り返しタスクを柔軟に設定できるし有料版も$2/月と安く，かなり使いやすいです。
特徴的だなと思った機能はゴールです。有料版のみですが，タスクを何の目的なのかというゴール単位でまとめることが可能です。

3. Todoist

仕事用に，Mac版アプリで4ヶ月ほど使いました。
無料版で十分使えましたが，繰り返しタスクの設定方法が分からずストレスを感じてました。(最近ようやく設定方法が分かりました。。)
面白いと思ったのはカルマという機能で，タスクを完了したりすることでカルマが貯まっていきます。
モチベーションを維持しやすくするためのしくみが有り，継続しやすいと感じました。

4. Wunderlist

2016年になり，プライペートも疎かにしたくないと思い，仕事とプライベートのタスク管理ツールの統合を図りたいと思いました。
Todoistが辛くなってきたこともあり，UIがいけてる & 無料で充分使えそうと前から気になっていたWunderlistに移行してみます。

Android + Macのアプリで使って一週間くらい経ちましたが概ね良い感じですが，
何点か不便に感じ始めてます。

  • タスクのソートが手動なのが不便 (慣れの問題？)
  • 期限日の指定が 今日・明日・日付指定 のみで，来週がないのが不便

ある程度は慣れで解消していくと思っているので，タスクマネジメントツール探しの旅にそろそろ終わりを告げたいところです。
